</p>

## ⚙️ Установка
1. Скачайте архив с программой из раздела **Releases**.
2. Распакуйте в удобную вам папку.
3. Запустите `Manager.bat` и выберите цвет иконок.
4. Готово! Теперь по нажатию ПКМ в любом месте проводника Windows у вас появится пункт **"Скачать"**.

## 🚀 Как пользоваться
1. Скопируйте ссылку на видео в браузере.
2. Перейдите в папку, куда хотите сохранить файл.
3. Нажмите **ПКМ** на пустом месте -> **Скачать** и выберите нужный вариант:

* **Аудио** — извлечение дорожки в формате `.mp3`.
* **Видео** — загрузка в формате `.mp4`.
* **Без рекламы** — загрузка видео с автоматическим вырезанием интеграций (через **SponsorBlock**).
* **Отрывок** — загрузка фрагмента видео или аудио по таймкодам (начало и конец).
* **Превью** — сохранение обложки видео в высоком разрешении.

> ℹ️ **Примечание:** Все медиафайлы автоматически скачиваются в **максимально доступном качестве**.

## 🔧 Дополнительно
Если программа уже установлена, вы можете в любой момент снова запустить `Manager.bat`, чтобы:
* **Сменить цвет иконок**
* **Обновить yt-dlp** — скрипт автоматически проверит наличие новой версии и скачает её. (Старая версия будет перемещена в папку `old`)
